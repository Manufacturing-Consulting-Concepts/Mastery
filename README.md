## Mastery

Mastery is used to assist companies to prepping for a security audit.
It accomplished this by taking in a csv file of all of the compliance requirements
and choosing a control at random. 

Users can take notes on each control.  


## Custom Controls List

This tools comes default ready for CMMC / NIST SP-800 171.

Others can be added by import a new csv file that conforms to the structured
format required by the application.
